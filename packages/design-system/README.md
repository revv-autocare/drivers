# Design system

Shared tokens, icons, and component primitives across the driver, shop, and admin apps.

`styles/tokens.css` is the canonical source of truth for colors, type, radii, shadows, and spacing. Imported via `import "design-system/styles/tokens.css"` from each app's entry point.

Component primitives will be extracted from `apps/driver-prototype/` as the real apps are built.
